# HereNow

A mobile app that facilitates real, mutual connections between friends by matching availability for short chats.

## Concept

- **Signal availability** — Let your connections know when you have even 5 minutes for a quick catch-up
- **Mutual matching** — When two friends are both available, the app suggests a connection
- **Stay in rhythm** — Tracks when you last connected with each person so no one falls off your radar
- **Social, not parasocial** — Built for two-way relationships, not broadcasting
